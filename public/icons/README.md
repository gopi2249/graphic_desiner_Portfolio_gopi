# Icons Directory

Place your icon files in this directory:

- `email.svg` - Email icon (blue)
- `phone.svg` - Phone icon (blue)
- `location.svg` - Location/map pin icon (blue)
- `brand-identity.svg` - Brand identity icon (blue)
- `digital-design.svg` - Digital design/monitor icon (blue)
- `print-design.svg` - Print/printer icon (blue)
- `packaging.svg` - Packaging/box icon (blue)

All icons should be in SVG format and colored blue (#5776E5).
